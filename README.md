# Girls Mobile Carousel

Static mobile carousel widget for Readymag iframe embeds.

## Structure

- `index.html`
- `style.css`
- `script.js`
- `assets/models/`
- `assets/stats/`
- `assets/icons/`

## Cloudflare Pages

No build command is required. Use the project root as the output directory.

```html
<iframe
  src="https://girls-mobile-carousel.pages.dev/"
  style="width:100%; height:100vh; border:0; background:transparent;"
  allowtransparency="true"
></iframe>
```
